# Youtube-Clone
youtube의 프론트엔드의 기능 및 스타일들을 최대한 따라하면서 각 기능에 대한 것들에 대해서 공부하고, 백엔드는 잘 모르니 다양한 정보(책, 공식 문서, 유명 블로그, 유명 회사)를 참조하면서 구성해보자.
예상되는 프로젝트 진행 기간 : 1/30 ~ 3/10
